# OpenVASP

OpenVASP Typescript Library

## Getting OpenVASP Library

To install:

```
$ npm i --save openvasp
```

## Development

Instal dependencies

```
$ npm install
```

Build

```
$ npm run build
```

Test

```
$ npm test
```

To publish to NPM:

```
$ pika publish
```

To publish dosc:

```
$ npm run doc
```
